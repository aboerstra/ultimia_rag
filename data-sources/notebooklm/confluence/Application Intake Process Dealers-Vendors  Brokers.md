Next Steps</strong></h2><ul><li><p><strong>Referral Deal</strong>: If there is a Deal to be processed as referred by the new Dealer/Broker, a submission is made to <a href="mailto:submit@maximcc.com">submit@maximcc.com</a> and proceed with the Application Intake Process (Applicants).</p></li></ul>
